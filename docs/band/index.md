# About MJTH Band

**MJTH Band** คือกลุ่มเพื่อนที่เล่นดนตรีออนไลน์ด้วยกัน ตั้งแต่ปี 2021 ผ่านแพลตฟอร์มต่างๆ เช่น Clubhouse, Facebook, YouTube, JOOX และ TikTok
