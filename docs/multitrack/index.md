# ระบบบันทึกเสียงแบบ Multitrack

เซิร์ฟเวอร์ MJTH.live มีระบบบันทึกเสียงแบบ multitrack (แยกแทร็กแต่ละคน) โดยสามารถควบคุมได้ในโปรแกรม Jamulus

## ความแตกต่างระหว่าง Multitrack และ Clipper

| คุณสมบัติ            | Multitrack Recording                       | Clipper (Single-track)                       |
| -------------------- | ------------------------------------------ | -------------------------------------------- |
| **จำนวนแทร็ก**       | แยกแทร็กของแต่ละคนเป็นไฟล์ .wav ต่างหาก    | รวมเสียงทุกคนเป็น .mp3 ไฟล์เดียว             |
| **คุณภาพเสียง**      | Lossless (WAV, 48kHz)                      | Lossy (MP3, 128kbps)                         |
| **การใช้งาน**        | สำหรับโพสต์โปรดักชัน, แก้ไข, มิกซ์         | สำหรับฟังย้อนหลังเท่านั้น                    |
| **การเริ่มต้น**      | ต้องพิมพ์ `/start` เพื่อเริ่มบันทึก        | พิมพ์ `/on` แล้วใช้ `/clip` บันทึกทีหลัง     |
| **เวลาบันทึก**       | สูงสุด 10 นาที เมื่อครบแล้วจะหยุดอัตโนมัติ | เก็บย้อนหลัง 10 นาที โดยมีการบันทึกต่อเนื่อง |
| **ขนาดไฟล์**         | ใหญ่มาก (~100-200 MB ต่อ 10 นาที)          | เล็ก (~10 MB ต่อ 10 นาที)                    |
| **รองรับการเล่นซ้ำ** | ต้องดาวน์โหลดและเปิดใน DAW                 | เล่นผ่าน browser ได้ทันที                    |

## การใช้งานระบบ Multitrack Recording

สามารถควบคุมระบบบันทึกเสียงแบบ multitrack ได้ผ่านการพิมพ์คำสั่งในแชทของ Jamulus:

| คำสั่ง   | คำอธิบาย                       |
| -------- | ------------------------------ |
| `/start` | เริ่มบันทึกเสียงแบบ multitrack |
| `/stop`  | หยุดการบันทึกก่อนครบเวลา       |

### `/start`

- เมื่อสั่ง `/start` แล้ว ระบบจะเริ่มบันทึกเสียงทันที
- ระบบจะบันทึกเสียงได้สูงสุด **10 นาที** หลังจากนั้นจะหยุดอัตโนมัติ
- เสียงแต่ละคนจะถูกบันทึกเป็นไฟล์ `.wav` แยกกัน พร้อมกับไฟล์ `.lof` (Audacity project file) สำหรับเปิดใน Audacity และไฟล์ `.rpp` (Reaper project file) สำหรับเปิดใน Reaper

### `/stop`

- เมื่อสั่ง `/stop` แล้ว ระบบจะหยุดการบันทึกทันที (ไม่ต้องรอ 10 นาที)
- ระบบจะอัปโหลดไฟล์บันทึกไปยังเซิร์ฟเวอร์และส่งลิงก์ดาวน์โหลดผ่านแชท
- หากไม่สั่ง `/stop` ระบบจะหยุดอัตโนมัติเมื่อครบ 10 นาที

## ไฟล์ที่ได้จากการบันทึก

เมื่อการบันทึกเสร็จสิ้น ระบบจะสร้างไฟล์ `.zip` ที่ประกอบด้วย:

- **ไฟล์ `.wav`** - แต่ละคนจะมีไฟล์เสียงของตัวเอง (48kHz, 16-bit stereo WAV)
- **ไฟล์ `.lof`** - Audacity project file ที่สามารถเปิดใน Audacity เพื่อแก้ไขและมิกซ์ได้เลย
- **ไฟล์ `.rpp`** - Reaper project file ที่สามารถเปิดใน Reaper เพื่อแก้ไขและมิกซ์ได้เลย
