# กลุ่มผู้ใช้งาน Jamulus ในไทย

## Thailand Jamulus User Group

**Thailand Jamulus User Group** เป็นกลุ่มที่รวบรวมนักดนตรีชาวไทยที่สนใจเล่นดนตรีร่วมกันผ่าน Jamulus สามารถเข้าร่วมกลุ่มผู้ใช้งาน Jamulus ในไทยได้ทาง Discord และ Facebook:

- [เข้ากลุ่ม Discord](https://discord.gg/tZ8bKmsrcp)
- [เข้ากลุ่ม Facebook](https://www.facebook.com/groups/jamulus.th/)

::: tip

หากมีปัญหาการใช้งาน ขอแนะนำให้[เข้ามาคุยกันในกลุ่ม Discord](https://discord.gg/tZ8bKmsrcp)

:::

## MJTH.live

**MJTH (MusicJammingTH)** เป็นเซิร์ฟเวอร์ Jamulus สาธารณะเซิร์ฟเวอร์แรกในประเทศไทย ที่เปิดให้บริการอย่างต่อเนื่องตั้งแต่ปี 2021 [อ่านรายละเอียดเพิ่มเติม &rarr;](/about/)

## Dharma Room

**Dharma Room** หรือ **ห้องสอนแล้วทำมั้ย** เป็นเซิร์ฟเวอร์ Jamulus สาธารณะที่เปิดให้บริการอย่างต่อเนื่องตั้งแต่ปี 2022 เซิร์ฟเวอร์ดูแลโดยคุณ Ace ในกลุ่มมีการอัดเพลงเป็นครั้งคราว สามารถติดตามข้อมูลเพิ่มเติมได้ที่ [เพจ “แจมดนตรี ห้องสอนแล้วทำมั้ย”](https://www.facebook.com/profile.php?id=100092503326040) และชมคลิปดนตรีทาง [YouTube @DharmaRoomMusic](https://www.youtube.com/@DharmaRoomMusic)

## JamulusTH.com

**JamulusTH.com** เป็นเซิร์ฟเวอร์ Jamulus สาธารณะ ที่เปิดให้บริการอย่างต่อเนื่องตั้งแต่ปี 2023 เซิร์ฟเวอร์ดูแลโดยคุณ LAB และ[เว็บไซต์ JamulusTH.com](https://jamulusth.com/) “ก่อตั้งขึ้นเพื่อประชาสัมพันธ์ และรวบรวมความรู้เกี่ยวกับการเล่น และซ้อมดนตรีออนไลน์ เพื่อเป็นแหล่งแลกเปลี่ยนเรียนรู้ทักษะ การเล่นดนตรี นักดนตรี และผู้ชื่นชอบการเล่นดนตรี”

## Stream ทาง Discord

นอกจากเซิร์ฟเวอร์หลักๆ ตามที่กล่าวมาข้างต้นแล้ว ในชุมชนยังมีการเชื่อมเสียงจากเซิร์ฟเวอร์ Jamulus ไปยัง Discord เซิร์ฟเวอร์ต่างๆ เช่นเซิร์ฟเวอร์ IBO Community และ Hakuu Community อยู่เรื่อยๆ เปิดเป็นห้องดนตรีสด นำโดยคุณ Petch ช่วยให้คนที่เล่น Discord ได้รู้จักกับการเล่นดนตรีสดแบบออนไลน์อีกช่องทางด้วย

## เซิร์ฟเวอร์อื่นๆ

โปรแกรม Jamulus เป็นซอฟต์แวร์เสรี ผู้ใช้งานสามารถเปิดเซิร์ฟเวอร์ขึ้นมาใช้งานได้เอง โดยการตั้งเครื่องของตัวเองเป็นเซิร์ฟเวอร์ รายชื่อเซิร์ฟเวอร์ที่เปิดให้บริการอยู่ในปัจจุบัน สามารถดูได้ที่หน้า [รายชื่อเซิร์ฟเวอร์ &rarr;](/servers/)
