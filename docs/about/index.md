# MJTH Servers

**MJTH Servers** คือเซิร์ฟเวอร์ Jamulus สาธารณะเซิร์ฟเวอร์แรกในประเทศไทย ที่เปิดให้บริการอย่างต่อเนื่องมาตั้งแต่ปี 2021 รองรับผู้ใช้งานพร้อมกันได้สูงสุด 40 คน

เซิร์ฟเวอร์ของ MJTH เปิดให้ใช้บริการฟรี โดยอาศัยการสนับสนุนจากผู้ใช้งาน คุณสามารถ[ร่วมสนับสนุนค่าใช้จ่าย](/support/)ในการเช่าเซิร์ฟเวอร์และดูแลรักษาระบบได้
