# About MJTH.live

**MJTH.live** เป็นเซิฟเวอร์ Jamulus สาธารณะเซิฟเวอร์แรกในประเทศไทย ที่เปิดให้บริการอย่างต่อเนื่องมาตั้งแต่ปี 2021 รองรับผู้ใช้งานพร้อมกันได้สูงสุด 40 คน
